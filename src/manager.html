A solução gratuita para gestão de reservas.

O que é?

Uma web app compatível com desktop ou tablet, que permite gerir reservas de forma simples e em qualquer lugar. É um canal de venda e uma ferramenta de marketing.

Para quem?

Para qualquer pessoa que tenha uma infra-estrutura desportiva e a queira rentabilizar. Clubes, Condomínios, Escolas, Universidades, Centros Desportivos, etc.

Como posso fazer parte?

Envie um mail para business@bookball.co e a nossa equipa faz o resto.
